Debian Personal Package Archive for cloud-initramfs-tools

Although the overlayroot present in Ubuntu helps protect RootFS, it is not possible to install the overlayroot package from the official Apt repository in Debian 10 and Debian 11.

Fortunately, however, the cloud-initramfs-tools maintainer team has released packages for Debian 10 and Debian 11 that include overlayroot.

This PPA repository is intended to use the work of the cloud-initramfs-tools maintainer team and redistribute the overlayroot package.

Supported platforms

  • Debian 10 (buster)
  • Debian 11 (bullseye)

Install required packages

sudo apt-get update
sudo apt-get install apt-transport-https ca-certificates

Download GPG public key

sudo wget -O /etc/apt/keyrings/ppa-debian-cloud-initramfs-tools.asc https://takumi.tmfam.com/ppa-debian-cloud-initramfs-tools/pubkey.asc

Configure APT sources.list

echo "deb [signed-by=/etc/apt/keyrings/ppa-debian-cloud-initramfs-tools.asc] https://takumi.tmfam.com/ppa-debian-cloud-initramfs-tools $(lsb_release -sc) main" | tee /etc/apt/sources.list.d/ppa-debian-cloud-initramfs-tools.list
echo "deb-src [signed-by=/etc/apt/keyrings/ppa-debian-cloud-initramfs-tools.asc] https://takumi.tmfam.com/ppa-debian-cloud-initramfs-tools $(lsb_release -sc) main" | tee /etc/apt/sources.list.d/ppa-debian-cloud-initramfs-tools.list
